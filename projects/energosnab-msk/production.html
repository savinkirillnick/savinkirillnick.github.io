( 01 ) Площадки

Две площадкиполного цикла

ПЛОЩАДКА 01

Волоколамск

Первая производственная площадка компании «ЭнергоСнаб», запущенная в 2016 году в Волоколамском районе Московской области. Основное направление — производство опор ЛЭП (стойки СВ) методом центрифугирования.

Возможности площадки:

  • Центрифугирование предварительно-напряжённых опор
  • Производство опор СВ всех типоразмеров
  • Камеры пропаривания для ускоренного набора прочности
  • Складские запасы для оперативной отгрузки
ПЛОЩАДКА 02

Воскресенск

Вторая производственная площадка открыта в 2020 году в городе Воскресенск Московской области. Расширение производственных мощностей позволило удвоить объёмы выпуска и обеспечить полный контроль качества на всех этапах.

Возможности площадки:

  • Расширенное производство ЖБИ для ЛЭП
  • Аттестованная лаборатория на территории
  • Полный цикл испытаний — от УЗК до нагрузочных тестов
  • Производство нестандартных изделий по чертежам

( 03 ) Оборудование и линии

Современноеоборудование

01

Линии центрифугирования

Производственные линии для центрифугирования предварительно-напряжённых опор ЛЭП. Технология обеспечивает идеальную геометрию и высокую плотность бетона марки М400-М500.

02

Бетоносмесительная установка

Автоматизированная бетоносмесительная установка с точной дозировкой компонентов. Контроль качества на каждом этапе приготовления смеси — от входного контроля цемента до готового бетона.

03

Оборудование преднапряжения

Стенды для натяжения арматуры с точным контролем усилия преднапряжения. Использование высокопрочной арматуры класса А-IV и выше с гарантией соблюдения ГОСТ требований.

04

Камеры пропаривания

Паровые камеры для тепловлажностной обработки изделий. Технология ускоренного набора прочности бетона с контролем температурного режима по ГОСТу для сокращения сроков изготовления.

05

Испытательное оборудование

Аттестованная лаборатория с оборудованием для УЗК (ультразвуковой контроль), прессами для испытаний на прочность, измерительными инструментами для контроля геометрии и трещиностойкости.

06

Складское хозяйство

Организованные складские площадки для хранения готовой продукции с системой учёта. Погрузочная техника — краны-манипуляторы для безопасной отгрузки изделий без повреждений.

( 04 ) Мощности

Производственныемощности

Наши производственные площадки оснащены современным оборудованием и способны обеспечить потребности крупных инфраструктурных проектов любого масштаба.

Две площадки позволяют нам гибко управлять производственным процессом, равномерно распределять нагрузку и гарантировать стабильные сроки изготовления даже при пиковых заказах.

Складские запасы обеспечивают оперативную отгрузку востребованных типоразмеров опор и фундаментов — без ожидания производства.
